Project Description

This project focuses on building an Intrusion Detection System (IDS) using multiple machine learning classifiers. The aim is to detect potential cyber-attacks by analyzing structured data containing system/network features.

Multiple ML algorithms are trained and compared, and the XGBoost classifier achieved the highest accuracy, making it the best-performing model for predicting whether an attack has occurred.

The project covers the full ML pipeline, starting from dataset ingestion to final prediction and model saving.

Key Features of This IDS

End-to-end automated ML pipeline

SMOTE applied to balance the dataset

Label Encoding applied to categorical features

Standard scaling applied to numerical features

Multiple ML models trained and compared

Model persistence using .pkl files

Feature importance visualization

Interactive prediction using user-provided inputs
